# ngrok 部署记录

## 时间
2026-06-20 19:32

## 目标
将小酥存货展示网站（斗罗大陆H5）通过 ngrok 部署为公网可访问链接

## 操作步骤
1. 下载 ngrok v3 stable Windows 二进制到 xiaosu-site/tools/ngrok.exe
2. 用户提供了 ngrok authtoken: 3FOneY...fYxS
3. 执行 `ngrok config add-authtoken` 保存配置
4. 启动隧道: `ngrok http 18888`（本地 Node HTTP 服务运行在 18888 端口）

## 结果
- **公网链接**: https://shale-scone-coveted.ngrok-free.dev
- 无拦截页，直接访问
- 依赖：本地电脑开机 + Node HTTP server 运行 + ngrok 进程运行

## 备用链接（localtunnel）
- https://xiaosu888.loca.lt（需输入IP绕过拦截页）

## 关键文件
- C:\Users\没饭局\.qclaw\workspace\xiaosu-site\index.html（网站文件）
- C:\Users\没饭局\.qclaw\workspace\xiaosu-site\tools\ngrok.exe（ngrok二进制）
- C:\Users\没饭局\AppData\Local\ngrok\ngrok.yml（ngrok配置含authtoken）

## 注意事项
- ngrok 免费版域名随机，每次重启会变
- 如需固定域名，需要升级 ngrok 付费计划
- 本地服务(18888)和ngrok进程都需要保持运行
